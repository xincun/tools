About Thunder
====================
    Thunder is a new breed download manager basing on cutting-edge Grid Computing technology. As it is especially optimized for broad band users, it could fully utilize the bandwidth and bring end user a brand-new downloading experience. 

Features
====================
1．	Ultra-fast download speed basing on superior Grid Computer technology
2．	Strong task management feature to help you organize your downloads
3．	Intelligent Caching System to protect your hard disk
4．	Messaging System to make it more easy to use
5．	Proprietary Diagnostic System to help you detect and solve problems
6．	Seamless integration with Anti-Virus software to make your system virus free
7．	Automatic online upgrade
8．	Different kinds of skins to choose from
10．	Multiple languages support
11．	Data integrity check and correction
12．	Proxy server support
13．	Detailed resource and connection information to make you aware the download status
14．	FTP site explorer support
15．	Supporting download history import/export, easy to share your lists with friends
16．	Rich configuration items to help you customized your download manager

System Requirement
====================
* Win98/Me/2000/XP/2003
* 128MB Memory or more
* 100MB Hard Disk。
* 800X600 resolution or higher

Installation
====================
    After you obtain the Thunder Installer, double click it and proceed to install as instructed.

Upgrade
====================
1．	Thunder will prompt you to upgrade once it detects a newer version is available
2．	You can also download the latest version from Thunder's website and install it as instructed

Uninstall
====================
1．	You can remove it from Start - All Programs - Thunder Networking - Uninstall Thunder.
2．	You can remove it from "Control Panel" - "Add/Remove Programs".

Usage
====================
    Right click in browser the URL you want to download, and choose "Download by Thunder", a dialog box will show up, and you can choose the directory to save and press "OK".

迅雷
====================
    迅雷是一款新型的基于多资源超线程技术的下载软件，作为“宽带时期的下载工具”，迅雷针对宽带用户做了特别的优化，能够充分利用宽带上网的特点，带给用户高速下载的全新体验！同时，迅雷推出了“智能下载”的全新理念，通过丰富的智能提示和帮助，让用户真正享受到下载的乐趣。

特点
====================
1．	全新的多资源超线程技术，显著提升下载速度；
2．	功能强大的任务管理功能，可以选择不同的任务管理模式；
3．	智能磁盘缓存技术，有效防止了高速下载时对硬盘的损伤；
4．	智能的信息提示系统，根据用户的操作提供相关的提示和操作建议；
5．	独有的错误诊断功能，帮助用户解决下载失败的问题；
6．	病毒防护功能，可以和杀毒软件配合保证下载文件的安全性；
7．	自动检测新版本，提示用户及时升级；
8．	提供多种皮肤，用户可以根据自己的喜好进行选择；
9．	可以限制下载速度，避免影响其他网络程序；
10．	支持多种语言；
11．	独有的文件校验功能，保证下载文件的完整性；
12．	完善的代理设置，允许不同的连接使用不同的代理；
13．	详尽的资源信息和连接信息，帮助用户更好的了解下载状态；
14．	FTP资源探测器，配合迅雷使用可以方便的下载FTP上的文件；
15．	导入/导出下载列表，可以方便的和朋友分享下载列表；
16．	丰富的配置项，可以根据自己的需要进行个性化的设置。

系统要求
====================
* Win98/Me/2000/XP/2003
* 128MB 内存或更多。
* 100MB 硬盘空间。
* 800X600 分辨率或以上

如何安装
====================
    获得迅雷的安装文件后，鼠标左键双击该安装文件，按照提示安装即可。

如何升级
====================
1．	迅雷会自动检测是否存在新版本时，当存在新版本时会自动提示用户是否升级到最新版本；
2．	用户也可以下载最新版本的安装文件，鼠标左键双击该安装文件，按照提示安装即可升级到最新版本。

如何卸载
====================
1．	点击开始菜单，通过迅雷程序项中的卸载迅雷快捷方式卸载迅雷；
2．	打开控制面板，在添加或删除程序中卸载迅雷。

如何使用
====================
    鼠标右键点击需要下载的文件地址，选择“使用迅雷下载”，在弹出的新建任务面板中设置存放目录后点击确定按钮即可。
